\chapter{Introduction} \label{ch:introduction}

The Interstage Interface Panels (IIP) are three vertical mounted PCBs designed 
to give external access up to four modules inside of a 2U CubeSat during 
final assembly, integration and testing (AIT) before launch.
The complete set of the boards allow the nanosatellite to be charged, programed and debugged. The usage of this hardware platform is taking into account the use of a MSP-FET: MSP430 Flash Emulation Tool from Texas Instruments for JTAG interfacing and debugging, UART debugging through a mini USB B port interfacing the FT4232H USB bridge IC from FTDI, a JST XH series connector for charging internal batteries and a Remove Before Flight (RBF) pin header.
