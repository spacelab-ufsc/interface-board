# Semi USB Interface Board
Directory for files of the Semi USB variant of Interface Board 2.0

The Semi USB Interface Board is composed by 4 pcbs integrated on the four sides of the FloripaSat-2 2U Cubesat structure.

The "Source of all boards" stores the altium files for each pcb that is within its own project.

The "Output files of all boards" stores the PDF, BOM, Gerber, NC Drill, ODB++ and Output Job Files for each project.

All pcbs will be assembled in "Multi Board Assemby" directory.
